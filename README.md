Race Series Manager – WordPress Plugin

Race Series Manager is a custom WordPress plugin designed to manage endurance events such as running races, cycling races, triathlons, and multi-day race series. It provides organizers with a complete backend system for creating, displaying, and managing races and events within WordPress, offering a flexible and scalable solution for both single races and complex series.

Key Features

Race & Event Management
Create events and add multiple races under each event. Each race can include detailed information such as date, distance, elevation, location, and custom metadata.

Results Integration
Add results via iframe or external links. Results automatically appear as dedicated buttons on race pages and event overview pages.

Registration Support
Provide registration functionality through either iframe embeds or external URLs. Registration buttons are dynamically generated on both race and event pages.

Participants View
Display participant lists using iframe embeds or external links, with automatic button placement on the race and event front-end.

Live Updates
Integrate live timing or tracking via iframe or URL, shown as a “Live” button on race and event pages.

Custom Meta Boxes
Intuitive backend fields for all race details, iframe codes, and external URLs, making race setup simple and efficient.

Template-Friendly Structure
Designed to work with custom themes, blocks, and templates for full front-end customization.
