Race Series Manager is a lightweight WordPress plugin for organizing and displaying endurance events such as running, cycling, and multi-stage races. It lets you create events, add multiple races, and easily embed registration forms, results, participant lists, and live timing via iframe or external links.

Features

Create and manage events with multiple races

Add race details (distance, date, elevation, type, etc.)

Embed registration, results, participants, and live pages

Automatic buttons on race and event pages

Simple meta boxes for clean backend management

Theme-friendly frontend output

Perfect for race organizers, sports clubs, and event websites needing a simple but flexible race-management solution.
